# Compare String versions
* A python library for comparing 2 strings

> in case str1 = str2 `1 `
> in case str1 > str2 `0`
> in case str2 < str2 `-1`

#### Installation

> `pip3 install compare_strings_lib`

#### Usage
```
from strings_lib.compare import compare

    compare_strings = compare("1.2", "1.1")

```
